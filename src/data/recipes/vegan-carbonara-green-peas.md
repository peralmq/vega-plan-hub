## Ingredients

| quantity | unit | key             | ingredient                        | notes                       |
|----------|------|-----------------|-----------------------------------|-----------------------------|
| 2        |      | onion           | yellow onions                     |                            |
| 1        | tsp  | smoked-paprika  | smoked paprika powder             |                            |
| 4        | dl   | oat-cream       | oat cream                         | or other plant cream        |
| 1        |      | veg-bouillon    | vegetable bouillon cube           |                            |
| 600      | g    | green-peas      | frozen green peas                 |                            |
| 4        | port | pasta           | pasta                             |                            |
|          |      | salt            | salt                              | to taste                   |
|          |      | black-pepper    | black pepper                      | to taste                   |
|          |      | oil             | oil                               | for frying                  |
|          |      | vegan-cheese    | vegan cheese                      | optional, for topping       |

## Instructions

1. Start boiling water for the pasta.
2. Finely chop the onions. Heat oil in a pan and sauté the onions with smoked paprika until very soft.
3. Add oat cream and crumble in the bouillon cube. Add the peas.
4. Cook the pasta according to package instructions.
5. When the pasta is almost done, drain and add it to the sauce. Let it finish cooking in the sauce.
6. Season with salt and plenty of black pepper. Serve with vegan cheese if desired. 🍝🌱

## Notes

- Quick, creamy, and delicious weekday pasta! ✨
- Smoked paprika adds a lovely flavor.
- Top with extra peas or herbs.
- Great for meal prep.
