## Ingredients

| quantity | unit | key           | ingredient                        | notes                       |
|----------|------|---------------|-----------------------------------|-----------------------------|
| 500      | g    | spinach       | fresh spinach                     | or same amount frozen, thawed|
| 2-3      | dl   | water         | water                             | as needed for blending      |
| 1        |      | green-chili   | green chili, finely chopped       |                            |
| 2        |      | garlic        | garlic cloves, finely chopped     |                            |
| 2        | tbsp | ginger        | fresh ginger, chopped             |                            |
| 0.5      | dl   | oil           | neutral oil                       | for tempering              |
| 1        | tsp  | cumin-seeds   | whole cumin seeds                 |                            |
| 1        |      | onion         | onion, finely chopped             |                            |
| 0.5      | tsp  | turmeric      | turmeric                          |                            |
| 0.5      | tsp  | chili-powder  | chili powder                      |                            |
|          |      | hing          | hing/asafoetida                   | optional                   |
| 0.5      | tsp  | garam-masala  | garam masala                      |                            |
| 1        | tsp  | salt          | salt                              |                            |
| 400      | g    | paneer        | paneer or firm tofu, cubed        |                            |
|          |      | rice          | cooked rice                       | to serve                   |
|          |      | naan          | naan or roti/chapati              | optional                   |

## Instructions

1. Add spinach, chili, garlic, ginger, and some of the water to a blender. Blend to a smooth, thick purée, adding just enough water to blend.
2. Heat oil in a pan, fry cumin seeds until fragrant. Add onion and sauté until soft. Add turmeric, chili powder, hing, and garam masala, and fry for a minute.
3. Add the spinach purée, mix well, and simmer for about 10 minutes until thick. If too loose, simmer longer to reduce. Season with salt.
4. Fry paneer or tofu cubes until golden. Stir into the spinach sauce.
5. Serve hot with rice, naan, or roti/chapati. 🌱🧀

## Notes

- For vegan: use tofu instead of paneer.
- For extra flavor, add a splash of lemon juice before serving.
- Adjust chili to taste for more or less heat.
- This dish is naturally gluten-free and can be made fully vegan!
